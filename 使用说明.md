# Video to Script 使用说明

> 🎬 视频转剧本工具 — 一键从视频提取剧本结构、钩子分析和人物图谱

---

## 功能简介

| 功能 | 说明 |
|------|------|
| 语音转写 | 自动识别视频中的对白，带时间戳显示 |
| 背景音乐识别 | 区分对白与BGM，识别相似/准确的背景音乐名称 |
| 角色标注 | 自动标注每句台词的说话人 |
| 场景切割 | 智能检测场景变化，提取场景截图 |
| 钩子分析 | 分析视频中的8种钩子类型（开场悬念/反转/证据/倒计时/情感/恐惧/身份/呼应） |
| 结构化剧本 | 按场景拆解，输出画面描述+对白+情绪标注 |
| 人物图谱 | 分析人物关系网络，标注关系张力 |
| 改写建议 | 提供5种改写方向和付费节点建议 |
| 视频链接下载 | 支持抖音/YouTube/B站等1000+平台链接直接分析 |

## 安装步骤

### 前提条件

- **macOS 12.0+**（Monterey 或更高版本）
- **Python 3.9+**
- 约 **2GB 磁盘空间**（用于安装 AI 模型和依赖）

### 安装 Python（如果还没有）

**方法一：从官网安装（推荐）**

1. 访问 https://www.python.org/downloads/
2. 下载最新版 Python 3.x 安装包
3. 双击安装，一路「继续」即可

**方法二：通过 Homebrew 安装**

```bash
brew install python3
```

### 安装 ffmpeg（可选但推荐）

ffmpeg 可以加速音频提取和场景截图：

```bash
brew install ffmpeg
```

> 如果不安装 ffmpeg，工具会使用内置的 ffmpeg（速度稍慢）

## 启动方式

### 方式一：双击启动脚本（最简单）

1. 将 `video-to-script-app` 文件夹放到你想要的位置
2. 打开终端，运行一次以下命令（首次需要授权）：

```bash
chmod +x /path/to/video-to-script-app/run.sh
```

3. 之后每次双击 `run.sh` 即可启动

> 如果双击 `.sh` 文件无法启动，请右键 → 选择「打开方式」→「终端」

### 方式二：通过终端启动

```bash
cd /path/to/video-to-script-app
./run.sh
```

### 首次启动

首次运行会自动：
- 创建 Python 虚拟环境
- 安装所有依赖包（约需 3-5 分钟）
- 下载 Whisper 语音识别模型（约 150MB）

之后再次启动只需几秒。

## 使用指南

### 1. 配置 AI 分析（首次使用必须）

点击右上角 **⚙ 设置**，填写以下信息：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **API Key** | AI 服务的密钥 | `sk-xxxxx` |
| **API 地址** | AI 服务地址（国内用户必填） | `https://api.deepseek.com/v1` |
| **模型** | 使用的 AI 模型 | `deepseek-chat`（推荐） |

**推荐配置（国内用户）：**
- API Key: 你的 DeepSeek API Key（从 https://platform.deepseek.com/ 获取）
- API 地址: `https://api.deepseek.com/v1`
- 模型: `deepseek-chat`

> ⚠️ 如果不配置 API Key，仅能使用语音转写和场景检测功能，无法使用钩子分析、剧本分析等 AI 功能

### 2. 导入视频

**方式一：本地文件**
- 切换到「📁 本地文件」模式
- 拖拽视频文件到应用窗口，或点击「选择文件」
- 支持 MP4 / MOV / MKV / AVI / WEBM 格式

**方式二：视频链接**
- 切换到「🔗 视频链接」模式
- 粘贴视频链接（抖音/YouTube/B站/快手等）
- 点击「预览」查看视频信息，点击「下载」下载视频
- 下载完成后点击「开始分析」

### 3. 开始分析

点击 **▶ 开始分析**，工具将依次完成：

1. 提取音频 → 2. 语音转写 → 3. 场景检测 → 4. AI 分析

> 分析时间取决于视频长度，通常 1-3 分钟视频约需 30-60 秒

### 4. 查看结果

分析完成后，底部会显示以下标签页：

| 标签 | 内容 |
|------|------|
| **转写文本** | 带时间戳的台词，区分角色和BGM，显示背景音乐识别结果 |
| **场景切割** | 场景列表+截图（点击可放大） |
| **钩子分析** | 8种钩子类型分析（双击放大查看） |
| **结构化剧本** | 按场景拆解的完整剧本（双击放大查看） |
| **人物图谱** | 人物关系网络（双击放大查看） |

### 5. 导出结果

点击底部按钮导出：
- **📄 导出 Markdown** — 生成 `.md` 格式报告
- **📝 导出 Word** — 生成 `.docx` 格式报告（含角色颜色标注、场景截图）

## 设置说明

### 语音转写

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| Whisper 模型 | `base` | tiny最快但最差 | base平衡 | small推荐但慢 |
| 视频语言 | 自动检测 | 如果转写不准确可手动选择 |

### 场景检测

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 灵敏度 | 35（推荐） | 短剧推荐35，电影推荐25-30 |

### 视频下载

| 设置 | 说明 |
|------|------|
| Cookie 文件 | 部分平台需要登录才能下载高清，可从浏览器导出 Cookie |

## 常见问题

### Q: 启动后一直显示「分析中」怎么办？
A: 请确认已正确配置 API Key 和 API 地址。如果是国内网络，请使用 DeepSeek API 并填写 `https://api.deepseek.com/v1`。

### Q: 分析速度慢怎么办？
A: 
- 语音转写：使用 `tiny` 或 `base` 模型可以加速
- AI 分析：网络延迟可能导致慢，建议使用国内 API（如 DeepSeek）
- 场景检测：调高灵敏度阈值（如40或50）可以减少场景数量

### Q: 背景音乐没有被识别怎么办？
A: BGM 识别依赖 AI 推断，对于纯音乐或罕见曲目可能无法准确识别。工具会给出风格相似的候选歌曲。

### Q: 拖拽文件无法进入应用怎么办？
A: macOS 可能限制了拖拽权限。请尝试：
1. 使用「选择文件」按钮
2. 或在系统设置→隐私与安全性中授权终端访问文件

### Q: 链接下载失败怎么办？
A: 
- 确认链接是否可以正常打开
- 部分平台需要 Cookie，请在设置中配置
- 尝试更新 yt-dlp：`pip install -U yt-dlp`

### Q: 如何卸载？
A: 直接删除 `video-to-script-app` 文件夹即可。设置文件保存在 `~/.video-to-script-settings.json`，可手动删除。

## 技术架构

```
视频文件/链接
    ↓
音频提取 (ffmpeg)
    ↓
语音转写 (OpenAI Whisper)
    ↓
场景检测 (PySceneDetect)  ←→  AI 分析 (OpenAI API × 5 并行)
    ↓                              ├── 钩子分析
    ↓                              ├── 结构化剧本
    ↓                              ├── 人物图谱
    ↓                              ├── 改写建议
    ↓                              └── 角色标注 + BGM识别
    ↓
结果展示 & 导出 (Markdown / Word)
```
